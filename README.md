\# P2PFil


P2PFil is a privacy-focused peer-to-peer LAN file sharing and encrypted messaging application for Windows and Android.


\## Features V9.0.6.0

\- 📁 Peer-to-peer LAN file sharing with built-in Path Traversal defense to sandbox all remote requests.

\- 🔒 AES-GCM encrypted messaging with zero-allocation memory handling optimized for C# 12.0.

\- 🛡️ Replay Attack countermeasures using unique, atomic transaction packet identifiers (`MessageId`).

\- 🧠 Messages are stored in RAM only.

\- ✔️ Cryptographic file integrity verification using secure SHA-256 hashing.

\- ⏯️ Resume interrupted file transfers with robust, null-safe background worker controls.

\- 👥 Multiple simultaneous downloads with optimized ArrayPool memory management.

\- 🌐 No internet connection required.

\- 🖥️ Windows \& Android support with strict null-conditional platform runtime resiliency.



\## Requirements

\- Windows or Android

\- Same local network (LAN)
