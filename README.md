\# P2PFil



P2PFil is a privacy-focused peer-to-peer LAN file sharing and encrypted messaging application for Windows and Android.



\## Features



\- 📁 Peer-to-peer LAN file sharing

\- 🔒 AES-CTR encrypted messaging

\- 🧠 Messages are stored in RAM only

\- ✔️ File integrity verification

\- ⏯️ Resume interrupted file transfers

\- 👥 Multiple simultaneous downloads

\- 🌐 No internet connection required

\- 🖥️ Windows \& Android support



\## Requirements



\- Windows or Android

\- Same local network (LAN)
